# IMY 即时聊天 为私域而生

## 项目介绍
IMY 一键部署、数据自主、畅聊无阻

## 项目预览
web：https://imy.1xmb.cn


#特性
轻量级
高性能
所有后端技术模块纯Golang实现  
WebRTC技术实现音视频通讯


## 特点
一键部署
系统运行需要的文件，环境通过自动化下载和安装完成。让每个人都可以通过点击鼠标就能完成

数据自主
拥有数据库超管权限。自由备份、导入、迁移

畅聊无阻
imy是全WEB技术开发，可以在手机、电脑浏览器上实现跨平台文字、图片、声音、视频全模态的畅通聊天



##功能模块
登陆、随机获取用户登陆
发送表情+文字组合的富文本内容
发送图片内容，查看大图
enter发送信息，enter+ctrl换行输入内容
消息提醒
未读消息标记
记录历史会话
记录历史聊天内容
切换主题
发送视频
发送语音（仅移动端具备）
社区发帖、点赞、评论、回复
